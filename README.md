# 说明

>这是一个模仿喜马拉雅官网的个人学习项目。

>主要技术：Vue, Vue-Router, Vuex, axios, mock.js

>网址访问：[喜马拉雅官网模仿][1]

>谢谢您的观看 ^_^ 


## 项目运行
``` bash
# 克隆到本地
git clone https://github.com/ywAlexyw/ximalaya

# 进入文件夹
打开cmd命令

# 安装依赖
npm install

# 项目运行
npm run serve

# 发布环境
npm run build
```
